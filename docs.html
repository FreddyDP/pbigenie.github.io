Help / Docs

Last updated: 29 December 2025

Quick start

  1. Add the PBIGenie visual to your report (AppSource or imported .pbiviz, depending on your distribution).
  2. Drag a field into Category.
  3. Drag one or more measures into Value.
  4. (Optional) Drag a measure into Overlap to show the overlap layer.
  5. Open the Format pane to customize bars, markers, overlap, and labels.

Field well mapping

  • Category: the grouping field (e.g., Region, Product)
  • Value: one or more measures (e.g., Sales, Cost)
  • Overlap (optional): a measure used to render the overlap layer

Common tips

  • If your data contains positive and negative values, the visual will automatically handle the zero baseline.
  • Use the Overlap Bars formatting card to control thickness, color, and clamping behavior.
  • Use Overlap markers for head/marker customization and per-category marker colors.

Keyboard & accessibility

  • Use Tab to focus the visual.
  • Open the context menu using Shift + F10 (or the Menu key).
  • High Contrast is supported through Windows Accessibility themes.

Troubleshooting

  • Visual is blank: check that Category and at least one Value measure are populated.
  • Unexpected colors: verify per-category coloring toggles in the relevant formatting cards.
  • Overlap not showing: ensure the Overlap field is populated and “Show overlap” is enabled.
